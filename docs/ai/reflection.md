# Reflexión sobre el uso de IA
Una reflexión personal sobre el impacto de la IA en el aprendizaje de programación y en la eficiencia del trabajo diario.

# Implementación de Nuevas Funcionalidades 

Para mejorar la experiencia de usuario de TaskFlow, se han añadido 4 funcionalidades clave utilizando IA:

1. **Búsqueda Dinámica:** Implementación de un buscador que filtra el array de tareas mediante `.filter()` y `toLowerCase()`.
2. **Sistema de Filtros:** Botones de estado que permiten organizar la vista entre tareas activas y terminadas.
3. **Edición In-situ:** Uso de condicionales en el renderizado para intercambiar el texto por un campo de entrada (input).
4. **Gestión de Masa:** Función para eliminar todas las tareas completadas de una sola vez, optimizando el mantenimiento de la lista.

**Uso de la IA:** Se utilizó Cursor para generar la estructura lógica y asegurar que los nuevos elementos mantuvieran la coherencia estética (estilo lila) y la persistencia de datos en LocalStorage.
