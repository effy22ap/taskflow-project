# Diseño de mi primera App
Taskflow es una app para gestionar tareas de una manera limpia y sencilla.
Permite añadir, completar y eliminar tareas  , ademas de mostarte las estadisticas de como va tu progreso, asi puedes ver que tareas las has completado, cuales te faltan por terminar y el total de todas tus tareas.

## Estructura del proyecto
- `index.html`: UI principal (Tailwind vía CDN).
- `app.js`: lógica principal de la app (render, eventos, validaciones, LocalStorage).
- `script.js`: archivo de compatibilidad (stub) por si alguien aún lo referenciaba.
